# **App Name**: PetConnect

## Core Features:

- User Authentication: Secure user login and registration with session management using localStorage.
- Social Feed: Display a vertical scrolling feed with posts, stories, likes, comments, shares, and save interactions, designed in an Instagram-style format.
- AI Symptom Checker: Provide a multi-step AI tool to analyze pet symptoms and offer preliminary health advice, incorporating a reasoning tool.
- Lost & Found Alerts: Enable users to post and share missing pet alerts with generated images, location details, and contact information.
- Community Forums: Organize discussion forums by pet type (dogs, cats, etc.) with topics, replies, and user interactions.
- Local Events: Allow users to create and browse local pet-related events and meetups with RSVP functionality.
- Direct Messaging: Enable one-to-one direct messaging between pet owners, showcasing conversation lists and real-time updates.

## Style Guidelines:

- Primary color: Warm beige (#F0EBE4) to create a welcoming atmosphere.
- Background color: Light beige (#E8E3DB), a lighter variant of the primary, providing a soft, desaturated backdrop.
- Accent color: Earthy brown (#A88C6C), analogous to the primary hue, used sparingly to highlight interactive elements and maintain a natural feel.
- Body and headline font: 'Inter', a sans-serif font providing a modern, neutral and machined look.
- Use Lucide React icons, styled in brown to match the accent color and maintain visual consistency.
- Adopt a minimalist card-based layout for posts and events with a clean design inspired by Instagram.
- Implement subtle animations (e.g., fade-ins, transitions) to enhance user interaction and provide a polished feel.
